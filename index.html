À propos

Biblos

Une bibliothèque personnelle immersive rendue en Three.js. Les livres sont disposés sur des étagères en bois 3D avec ombres et matériaux réalistes. Cliquez sur un livre pour voir ses détails.

Données

books.json

Tous vos livres sont stockés dans books.json à la racine du projet. Exportez-le et rechargez la page pour persister. Le bouton Publier sur GitHub le pousse directement via l'API GitHub.

ISBN Lookup

Open Library

Entrez un ISBN-13 dans le formulaire d'ajout et cliquez Chercher pour pré-remplir titre, auteur, genre et couverture depuis Open Library.

Rendu 3D

WebGL / Three.js

Chaque livre est un BoxGeometry avec 6 matériaux distincts — couverture avant, dos, tranche, pages. Les textures sont chargées via le proxy images.weserv.nl pour éviter les erreurs CORS.
